# hyper-monokai

A monokai theme for hyper.

![Screenshot](http://i.imgur.com/UjvquUv.png)

## Installation

Edit `~/.hyper.js` and add `hyper-monokai` to `plugins`:
```js
module.exports = {

  /*...*/

  plugins: [
    'hyper-monokai'
  ]

  /*...*/

};
```
